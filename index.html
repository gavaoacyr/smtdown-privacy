SmtDown – Política de Privacidade

Última atualização: março de 2026

1. Sobre o Aplicativo

SmtDown é um aplicativo de distribuição de arquivos por código de acesso. Ele permite que usuários autorizados baixem arquivos a partir de um servidor privado, mediante apresentação de um url.

2. Dados Coletados

O SmtDown não coleta, armazena nem compartilha dados pessoais dos usuários.

  • Código de acesso: inserido pelo usuário, enviado ao servidor apenas para validação. Não é armazenado em servidores de terceiros.
  • URL do servidor: configurada pelo usuário e armazenada localmente no dispositivo com criptografia (EncryptedSharedPreferences).
  • Token de sessão (JWT): gerado pelo servidor após autenticação do código, armazenado localmente com criptografia e nunca transmitido para terceiros.

3. Dados NÃO Coletados

O aplicativo não coleta:

  • Nome, endereço ou dados de identificação pessoal
  • E-mail ou número de telefone
  • Localização geográfica
  • Dados de uso ou análise (analytics)
  • Dados de câmera, microfone ou contatos

4. Permissões Utilizadas

  • INTERNET: necessária para comunicação com o servidor de download.
  • REQUEST_INSTALL_PACKAGES: necessária para instalar baixados pelo aplicativo (apenas Android 8+).
  • WRITE/READ_EXTERNAL_STORAGE: necessária para salvar arquivos baixados no armazenamento do dispositivo (apenas Android ≤ 9).
  • POST_NOTIFICATIONS: usada para exibir progresso de download em segundo plano (apenas Android 13+).

5. Compartilhamento de Dados

O SmtDown não compartilha qualquer dado com terceiros, anunciantes ou serviços de análise. A única comunicação de rede ocorre diretamente entre o aplicativo e o servidor configurado pelo usuário.

6. Segurança

Todos os dados de configuração (URL do servidor e token de autenticação) são armazenados localmente usando EncryptedSharedPreferences com criptografia AES-256, respaldada pelo Android Keystore.

7. Menores de Idade

O SmtDown não é direcionado a menores de 13 anos e não coleta dados de crianças.

8. Alterações nesta Política

Qualquer atualização desta política será publicada nesta página. O uso contínuo do aplicativo após modificações indica aceitação da política revisada.
